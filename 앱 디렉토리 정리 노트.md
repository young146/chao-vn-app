# 앱 디렉토리 정리 노트

## PHP 파일 (WordPress 플러그인)

이 폴더에 있는 아래 PHP 파일들은 React Native 앱과 무관합니다.

| 파일 | 용도 |
|------|------|
| `chaovn-ad-api.php` | chaovietnam.co.kr WordPress 광고 API 플러그인 |
| `xinchao-company-directory.php` | chaovietnam.co.kr WordPress 업체 디렉토리 플러그인 |

### 배포 방법
- git commit/push 대상이 아님
- FTP로 서버에 직접 업로드: `wp-content/plugins/` 폴더에 배치

### 왜 여기 있나?
로컬 작업 편의상 같은 폴더에 두고 작업한 것. 앱 git 레포와는 별개.
