# 씬짜오베트남 디지털 마케팅 활성화 방안

> 작성일: 2026-05-18
> 전제: 디지털 인프라는 90% 갖춰져 있으나 트래픽(앱 설치 ~2,000)이 부족한 상태
> 참고: [BUSINESS_CONTEXT.md](BUSINESS_CONTEXT.md)

---

## 0. 한 줄 진단

> **인프라는 90% 갖춰져 있다. 부족한 건 트래픽이 아니라 *자산 간 전환 깔때기*다.**

이메일 7,000 / 앱 2,000 격차가 그 증거. **이메일 받는 사람의 70%가 아직 앱이 없다.** 외부 신규 유입을 늘리기 전에 *내부 자산 전환*만 풀어도 앱 1만 목표의 60~70%는 달성 가능.

---

## 1. 진단 — 무엇이 막혀 있는가

| 자산 | 보유 규모 | 활용 상태 | 손실 지점 |
|---|---|---|---|
| 매거진 24년 (2002~) | 베트남 한인사회 인지도 ★★★ | 디지털과 분리 | "신뢰 자산"이 앱으로 안 넘어옴 |
| 이메일 구독 | 7,000명 | 매일 발송, 측정·세그먼트 0 | 앱 설치 CTA 약함, 효과 미측정 |
| 카톡 오픈채팅 3개 | 잠재 N명 × 3 | 수동, 운영 비일관 | 카톡→앱 동선 없음 |
| 페이스북 페이지 | - | 수동 업로드 | 데일리 뉴스 자동 게시 안 됨 |
| 데일리 뉴스 파이프라인 | 자동 발행 | 자동화 끝 | 채널 다양화 안 됨 = 한계비용 0의 콘텐츠가 낭비됨 |
| jobs-crm AI 매칭 | 완성됨 | **미가동** | **USP(차별화 무기)가 잠자고 있음** |
| 광고주 풀 (잡지) | 검증된 BTB | 디지털 전환 미흡 | "잡지+앱 패키지" 영업 자료 부재 |

**핵심 통찰:** 시장 자체가 닫혀 있음 — 베트남 한인은 호치민 ~8만 + 하노이 ~3만 정도. 매스마케팅은 비효율, **커뮤니티 침투형이 압도적 효율**. 신규 채널 발굴보다 *기존 채널에서 끌어오기*가 ROI가 높다.

## 2. 3대 전략축

### A. 보유 자산 전환률 극대화 (단기, 0~30일)
이미 가진 7,000 이메일 + 3 카톡방 + 페북에서 앱으로 *끌어오는* 게 1순위.

### B. 한인 커뮤니티 침투 (중기, 30~90일)
신규 외부 유입은 *공식·반공식 한인 채널* 협력으로. 광고비 0원으로 도달 가능.

### C. 디지털 USP(고유 가치) 강화 (60~120일)
"왜 씬짜오 앱이어야 하는가" 답을 만든다. jobs-crm 매칭, 데일리 뉴스 푸시, 매거진 디지털 아카이브.

## 3. 액션 플랜 (30 / 60 / 90일)

### 첫 30일 — 자산 전환 깔때기 정비

| # | 액션 | 기대 효과 | 작업량 |
|---|---|---|---|
| 1 | **이메일 하단 고정 "앱 다운로드" 배너** (iOS/Android 버튼 + QR) | 7,000명 × 매일 노출. 월 200~500 신규 설치 가능 | 작음 (템플릿 수정) |
| 2 | **이메일 세그먼트 2분할**: "앱 미설치자 vs 설치자". 미설치자에게는 *주1회 앱 전용 콘텐츠* 미리보기 발송 | 전환률 추적·개선 시작 | 중간 (SendGrid 세팅 + DB 매칭) |
| 3 | **카톡 오픈채팅 3개 자동 게시 봇** — 매일 뉴스 카드 1장 + 앱 딥링크 1줄 자동 발송 | 수동 운영 부담 ↓, 노출 일관성 ↑ | 중간 (자동화 스크립트) |
| 4 | **페북 페이지 데일리 뉴스 자동 게시** (이미 카드 이미지가 생성됨 → Graph API 연결만) | 페북 도달 무료 확보 | 작음 |
| 5 | **jobs-crm 매칭 ON + 매일 매칭 이메일** | 구직자 *재방문 이유* 즉시 생성. 차별화 USP 1개 확보 | 중간 (기능 켜고 이메일 템플릿) |

### 31~60일 — 한인 커뮤니티 침투

| # | 액션 | 채널 |
|---|---|---|
| 6 | **호치민·하노이 한인회 / 한국학교 / 한인교회 공식 채널 협력 제안** — "베트남 한인 무료 디지털 인프라 후원" 명목, 매월 1회 뉴스레터 공동 발송 | 한인회 게시판/이메일/카톡공지 |
| 7 | **한인 식당·마트·미용실 매장 QR 스티커 배포** — "씬짜오 앱에서 이 가게 쿠폰" 명목으로 무료 광고 노출 + 앱 설치 유도. 잡지 광고주에게 우선 제공 | 오프라인 → 앱 다리 |
| 8 | **베트남 진출 한국기업 디렉토리** 를 미끼 콘텐츠로 LinkedIn/네이버블로그/주재원 카페에 노출 | "한국기업 1,000곳 검색" = 검색 트래픽 자석 |
| 9 | **추천 프로그램**: 친구 3명 앱 설치 시 *잡지 1년 정기구독권 무료* (한계비용 매우 낮음, 인지도 매우 높음) | 앱 내 / 이메일 |
| 10 | **카톡 오픈채팅 자체 성장 캠페인** — "씬짜오 부동산방" 등을 한인 커뮤니티 게시판에 노출 광고 (카톡방 = 앱 깔때기 입구) | 한인카페·블로그 |

### 61~90일 — USP 강화 & B2B 수익화

| # | 액션 |
|---|---|
| 11 | **광고주용 미디어킷 v1** 작성 — 앱 MAU, 이메일 도달, 카톡방 도달, 페북 도달 통합 리포트. "잡지+디지털 패키지 가격표". 능동적 영업 자료화 |
| 12 | **데일리 뉴스 다채널 자동 배포 파이프라인** 완성 — 1번 발행 → 앱/웹/이메일/카톡/페북/네이버블로그/한인카페 자동 전송 |
| 13 | **앱 푸시 알림 전략화** — 데일리 뉴스, 채용 신규, 부동산 신규를 사용자 관심사 기반 푸시 |
| 14 | **앱 첫 사용 온보딩 개선** — 첫 화면에서 *교민에게 즉시 유용한 무엇*(예: "오늘의 환율 + 베트남 뉴스 3개")이 즉시 보이도록. 첫 사용 retention의 90%가 첫 30초에서 결정 |
| 15 | **xinchao_crm + 이메일 통합** — 광고 문의자에게 *자동* 미디어킷 발송 |

## 4. KPI (마케팅 전문가가 매주 봐야 할 것)

| 지표 | 시작점 | 90일 목표 |
|---|---|---|
| 앱 누적 설치 | 2,000 | **4,500~5,000** (외부 유입 + 자체 자산 전환) |
| MAU | ? (측정 시작) | 설치자의 35%+ |
| 이메일 → 앱 전환률 | 0% (측정 안됨) | 8% |
| 이메일 오픈률 | ? | 25%+ (한인 대상 평균상) |
| 카톡 → 앱 클릭 | 0 (미측정) | 일 50건 |
| jobs-crm 매칭 발송 | 0 | 일 20건 |
| 광고주 신규 디지털 계약 | ? | 월 3~5건 |

**핵심:** 지금 측정 자체가 안 되는 항목들이 많음 → **첫 30일의 절반은 측정 인프라 세팅에 써야 함.** 안 그러면 무엇이 효과 있는지 영영 모름.

## 5. 만약 단 5개만 한다면 (Quick Wins)

> 인력·시간 제약을 가정한 *최소 실행 세트*. 30일 안에 끝낼 수 있고 효과 가장 큼.

1. **이메일 하단 앱 다운로드 배너 + QR** (2시간 작업)
2. **카톡 오픈채팅 데일리 뉴스 자동 게시 봇** (1~2일)
3. **jobs-crm AI 매칭 ON + 매일 이메일 발송** (2~3일)
4. **페북 페이지 자동 게시 연결** (반나절)
5. **GA / Mixpanel / Amplitude 셋 중 하나로 *측정 인프라 셋업*** (1~2일) — 안 하면 위 4개의 효과를 못 잼

이 5개만으로도 **90일 안에 신규 앱 설치 +1,500~2,500**, 이메일 효과 측정 시작, 카톡방-앱 동선 확립이 가능합니다.

## 6. 결정적 한 가지

> **씬짜오의 진짜 무기는 "신규 디지털 브랜드가 절대 가질 수 없는 24년 신뢰"다.**

대부분의 한인 대상 디지털 신규 진입자(부동산 앱, 구인구직 앱 등)는 신뢰 쌓는 데 1~2년이 걸립니다. 씬짜오는 *이미 있음*. 광고 문구·랜딩페이지·소셜포스트 어디에든 다음 메시지가 들어가야 합니다:

> "**2002년부터 베트남 한인과 함께해 온 씬짜오, 이제 앱에서 만나세요.**"

이 한 줄이 모든 디지털 마케팅의 헤드라인이 되어야 합니다. 광고비를 1원도 안 들이고 경쟁자 대비 *전환률을 1.5~2배* 끌어올릴 수 있는 자산입니다.
