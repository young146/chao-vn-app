## 📋 적용 방법

### Google Play Console
1. [Play Console](https://play.google.com/console) 접속
2. 앱 선택 → **스토어 등록정보** → **기본 스토어 등록정보**
3. 앱 이름, 짧은 설명, 전체 설명 붙여넣기
4. 저장 → 검토 제출

### App Store Connect
1. [App Store Connect](https://appstoreconnect.apple.com) 접속
2. 앱 선택 → **App 정보** (이름, 부제목, 키워드)
3. **버전 정보** (홍보 문구, 설명)
4. 저장 → 제출 (다음 앱 업데이트 시 적용)

> ⚠️ 홍보 문구(Promotional Text)는 앱 업데이트 없이도 즉시 변경 가능
