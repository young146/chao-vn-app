# 씬짜오 앱 개선 로드맵

개발+홍보 양면에서 정리한 우선순위 개선 항목.

---

## 1. ASO (앱 스토어 최적화) 🔴 진행 중
**목표**: Play Store / App Store 노출 및 설치율 향상  
**작업**: 키워드, 앱 설명, 스크린샷, 카테고리 최적화  
**난이도**: 낮음 | **임팩트**: 높음 (무료)

---

## 2. 오픈채팅 입장 환영 봇 ⬜ 대기
**목표**: 새 멤버 입장 시 자동으로 앱 링크 + 안내 발송  
**작업**: 카카오 오픈채팅 봇 API 연동  
**난이도**: 중간 | **임팩트**: 높음

---

## 3. 상품 공유 OG 이미지 개선 ⬜ 대기
**목표**: 카카오 공유 시 상품 실제 사진이 크게 노출  
**작업**: Cloud Function `viewItem`에서 OG 이미지를 상품 첫 사진으로 강화  
**난이도**: 낮음 | **임팩트**: 중간

---

## 4. 앱 내 "카카오로 공유" 버튼 ⬜ 대기
**목표**: 물건 상세에서 카카오 공유 → 딥링크 포함 카드 발송 → 바이럴  
**작업**: 카카오 SDK 공유 기능에 `/item/?id=` 딥링크 연결  
**난이도**: 낮음 | **임팩트**: 높음

---

## 5. 푸시 알림 다이제스트 ⬜ 대기
**목표**: "오늘 [도시] 새 매물 N개" 일일 digest push → 재방문율 증가  
**작업**: Cloud Function 스케줄러 + 도시별 집계  
**난이도**: 중간 | **임팩트**: 중간

---

## 6. 앱 설치 전환율 추적 ⬜ 대기
**목표**: 채팅방별 탭 URL 클릭 → 앱 설치 전환율 측정  
**작업**: Firebase Analytics 이벤트 또는 클릭 카운터 추가  
**난이도**: 낮음 | **임팩트**: 중간 (데이터 기반 의사결정)

## 7. 로그인 전환율 개선 ⬜ 대기
**목표**: 설치 → 로그인 전환율 35% → 50%+ (단기 목표)
**배경**: "방문자 모드"가 너무 강력해서 로그인할 동기 부족
**작업안 (택 1 또는 조합)**:
- 핵심 기능에 로그인 게이트 (당근 글 등록, 찜하기, 댓글 작성 등)
- 회원 전용 혜택 (매주 추첨 이벤트, 광고 게재 할인 등)
- 로그인 보너스 (광고 없는 뷰 1주일, 무료 게시 1회)
- 화면 곳곳에 "로그인하면 ___ 가능" 배너
**난이도**: 중간 | **임팩트**: 매우 높음 (마케팅 도달 인원 +75%)

---

## 8. 알림 권한 허용률 개선 ⬜ 대기
**목표**: 로그인 → 알림 허용 전환율 59% → 75%+
**배경**: 첫 실행 시 권한 팝업이 너무 빨리 떠서 거부됨
**작업안**:
- Pre-permission Modal: 권한 묻기 전에 "왜 알림이 필요한지" 설명 페이지
- 가치 있는 행동(찜하기/댓글/거래 시작) 시점에 권한 요청
- 거부 후 재요청 동선 (설정 진입 유도 배너)
**난이도**: 낮음~중간 | **임팩트**: 높음 (마케팅 도달 인원 +26%)

---

## 9. 매일 뉴스 + 홍보 푸시 자동화 ⬜ 대기
**목표**: 매일 정해진 시간에 "오늘의 뉴스" + 광고 카드 자동 푸시
**배경**: 광고/홍보 채널로 활용. 5번 항목(다이제스트)와 결합 가능
**작업안**:
- Cloud Functions Scheduler (예: 매일 오전 9시)
- WordPress REST API에서 최신 뉴스 1~3건 + 광고 카드 1건 조합
- 도시별/카테고리별 타깃팅 옵션
- 클릭률(CTR), 도달률 모니터링 대시보드
**난이도**: 중간 | **임팩트**: 매우 높음 (지속적인 트래픽 유도)
**선결 조건**: 항목 7, 8 개선 후 진행하면 ROI 극대화
