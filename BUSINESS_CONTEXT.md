# 씬짜오베트남 (Xin Chao Vietnam) — 사업 컨텍스트

> 이 문서는 Connect-AI 디렉토리에서 일하는 모든 AI 직원(서브에이전트)이
> 작업을 시작하기 전 반드시 읽어야 하는 회사 소개서입니다.
> 어조, 우선순위, 금기사항을 여기서 가져가세요.

---

## 1. 회사 한 줄 소개

**2002년부터 발행 중인 베트남 거주 한국인(교민) 대상 격주 오프라인 잡지사.**
최근 오프라인 광고 수익이 줄어드는 추세에 대응하여 **디지털 인프라**를 구축하고
**온라인 광고 + 부가 서비스**를 새로운 수익원으로 키우는 전환기에 있음.

---

## 2. 현재 운영 중인 디지털 자산

### 2.1 웹사이트
| 도메인 | 역할 |
|---|---|
| `chaovietnam.co.kr` | 잡지 홈페이지 + 데일리 뉴스 페이지 |
| `vnkorlife.com` | 교민생활 서비스 — 당근/나눔, 구인구직, 부동산 |

### 2.2 모바일 앱 (네이티브)
위 두 웹사이트를 네이티브로 래핑한 통합 앱.
탭 구성: **매거진 / 뉴스 / 당근·나눔 / 구인구직 / 부동산 / 이웃사업**

### 2.3 데일리 뉴스 파이프라인 (현재 핵심 마케팅 자산)
- 매일 아침: 베트남 현지 뉴스 + 한국 주요 뉴스 수집·요약
- 콘텐츠 생성: **`daily-news-final` 앱이 자동 발행 + 뉴스 카드 이미지 자동 생성**
- 발행 채널:
  - 웹사이트 + 앱 (자동)
  - **이메일 — SendGrid 로 자동 발송** (구독자 ~7,000명). 본문에 뉴스 카드 + 광고 카드(전기안전진단, 대열보일러 등) 자동 첨부
  - 페이스북 페이지 (현재 **수동 업로드**)
  - 카카오톡 오픈채팅방 (현재 **수동 업로드**, 이미지 광고 카드는 정책상 불가 — 텍스트 카드 1장만 첨부 가능)

### 2.4 자체 CRM 시스템 2종
- **xinchao_crm** (운영 중): 온라인 광고 문의 자동 수집 → 상담 단계별 기록 → 계약 관리
- **jobs-crm** (보유, 미가동): 구인 공고/구직자 DB. AI 매칭 기능 *만들어둠*. **현재 매칭은 돌리지 않고 데이터만 쌓는 중.** ← 미활용 자산

### 2.5 카카오톡 오픈 채팅방 (유입 깔때기)
- "씬짜오 구인구직"
- "씬짜오 부동산"
- "씬짜오 당근/나눔"

앱/웹과 연동되어 신규 사용자 유도용 입구로 사용.

## 3. 사업 목표 (우선순위 순)

1. **앱 설치자 10,000명 달성** (현재 약 2,000명) → 자연스러운 광고 수주 임계점
2. **온라인 광고 수익 채널 구축** — 줄어드는 오프라인 광고를 대체
3. **데일리 뉴스 이메일(7,000명) 을 능동적 마케팅 도구로 격상** — 현재 발송은 하지만 활용 미흡

---

## 4. 현재 가장 큰 병목 (AI 직원의 작업 우선순위)

| 병목 | 상태 | AI로 개선 가능성 |
|---|---|---|
| **홍보·마케팅 콘텐츠 생산** | 카톡·페이스북 수동 운영, 부족 | ⭕ 매우 높음 |
| **이메일 마케팅 전략화** | 7,000명에게 발송만 하고 있음, 세그먼트/효과 측정 부재 | ⭕ 매우 높음 |
| **jobs-crm 매칭 미가동** | 기능은 있는데 안 쓰는 중 | ⭕ 즉시 활성화 가능 |
| **앱 설치 유입 분석** | 어디서 막히는지 데이터 기반 진단 부재 | ⭕ 가능 |
| **광고주 영업 보조** | xinchao_crm 은 기록 중심, 능동적 영업은 사람 손 | △ 부분 가능 |

## 5. 어조·금기 (콘텐츠 작성 시 반드시 지킬 것)

- **대상**: 베트남 거주 한국인 (30~60대 교민, 주재원, 자영업자가 주류)
- **어조**: 친근하지만 잡지 매체로서의 신뢰감 유지. 인터넷 밈/유행어 남용 금지.
- **금지**:
  - 정치 편향 표현
  - 베트남 현지 비하·과도한 일반화
  - 한국·베트남 관계에 민감한 표현
  - 검증 안 된 정보·루머
- **항상**: 출처 명시, 단정 대신 인용형 표현
