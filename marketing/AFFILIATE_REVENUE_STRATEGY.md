> 작성일: 2026-05-28

> 전제: BUSINESS_CONTEXT.md + 디지털 마케팅 활성화 방안 기반
> 목표: 광고 수익에만 의존하던 구조 → 제휴 + 데이터 + 광고 + B2B의 4축 수익 모델

---

## 0. 전략 개요

### 0.1 4축 수익 모델
```
씬짜오 인프라 (7,000 이메일 + 2,000 앱 + 카톡 + 페북 + 잡지 24년)
  ├─ 1축: 잡지·앱 광고 (기존)
  ├─ 2축: 제휴 마케팅 ← NEW (이번 플랜)
  ├─ 3축: 데이터/리포트 판매 (광고주 미디어킷, B2B) ← 부수효과
  └─ 4축: 자체 유료 서비스 (비자/세무/송금 중개 등) ← 장기
```

### 0.2 단계별 원칙
- **Phase 1**: 가입만 하면 되는 제휴 (협상 불필요) → 데이터 축적
- **Phase 2**: Phase 1의 트래픽·전환 데이터로 협상 필요 제휴 접근
- **Phase 3**: 자체 데이터 자산을 미디어킷화 → 직접 영업

### 0.3 핵심 KPI
| 지표 | 90일 목표 |
|---|---|
| 제휴 가입 수 | 8개 이상 |
| 월 제휴 수익 | $500~1,500 |
| 위젯 클릭률 (CTR) | 5%+ |
| 위젯 전환률 (예약/가입) | 1%+ |
| 12개월 목표 월 수익 | $3,000~10,000 |

## Phase 0: 인프라 준비 (지금~7일)

협상 필요 없는 제휴를 시작하기 전에 **데이터를 측정할 수 있는 골격**을 먼저 세워야 합니다. 안 그러면 어떤 게 효과 있는지 모르고, 협상 단계에서 보여줄 카드가 없어요.

### Step 0-1. 추적 도메인 + UTM 표준화 (1일)

**왜**: 모든 제휴 링크가 어디서 어디로 클릭됐는지 알아야 함

**작업**:
- 단축 도메인 결정: `chaovietnam.co.kr/go/` 형태 → 모든 제휴 링크 통일
- WP 플러그인 또는 Vercel API 라우트로 리다이렉트 처리
- 클릭 로그 저장 (Firestore `affiliate_clicks` 컬렉션):
  ```js
  { partner: 'wise', source: 'email_20260528', campaign: 'phase1', 
    timestamp, user_id?, target_url, ip_country }
  ```
- 링크 형태: `chaovietnam.co.kr/go/wise?src=email&campaign=daily_20260528`
- 리다이렉트 시 UTM 자동 부착 + 제휴사 추적 파라미터 보존

### Step 0-2. 위젯 표시 영역 확정 (1일)

**5개 영역**에 통합 노출 전략:

| 영역 | 위치 | 형식 | 우선순위 |
|---|---|---|---|
| **A. 데일리 뉴스 이메일** | 본문 상단 또는 중간 | 자동 생성 이미지 + 텍스트 링크 | ★★★ |
| **B. 앱 홈 화면** | 뉴스 탭 상단 또는 별도 위젯 | 인터랙티브 카드 | ★★★ |
| **C. 앱 더보기 메뉴** | 새 섹션 "필수 서비스" | 카테고리별 그리드 | ★★ |
| **D. 웹 데일리뉴스 페이지** | 사이드바 또는 본문 | HTML 위젯 | ★★ |
| **E. 카카오톡 데일리 뉴스** | 본문 끝 1줄 (텍스트만) | "💱 환율은 이메일 확인 → chaovietnam.co.kr/go/X" | ★ |

### Step 0-3. 법적/투명성 표시 (0.5일)

베트남 거주 한국 사용자에게 신뢰 유지가 핵심. 다음을 명시:
- 모든 제휴 링크 옆에 작은 글씨 "*제휴 링크* — 클릭 시 씬짜오에 수수료가 지급됩니다"
- 별도 페이지 `/affiliate-disclosure` 작성
- 페북·카톡 게시물에도 첫 게시 시 명시

### Step 0-4. 위젯 디자인 시스템 (1~2일)

5개 영역 모두 같은 톤으로 통일된 위젯:
- 색상: 씬짜오 브랜드 색
- 폰트: 기존 데일리뉴스와 동일
- 차트 라이브러리: 이메일용 PNG는 QuickChart.io, 앱/웹은 Recharts
- 미니 스파크라인 + 큰 숫자 + 변동률 화살표

### Step 0-5. 자동 데이터 수집 파이프라인 (2일)

매일 새벽 자동 실행되는 Cron Job:
- 환율: ExchangeRate API
- 주가: 한국거래소 + TCBS
- 항공: Aviasales API (검색 결과 캐시)
- 임대료: vnkorlife 자체 데이터 + Batdongsan
- 결과를 Firestore `daily_data/{YYYY-MM-DD}` 문서로 저장

## Phase 1: 협상 불필요 제휴 가입 + 활성화 (7~30일)

가입 즉시 링크 받을 수 있는 제휴들. 사장님이 30분~1시간이면 신청 가능.

### 1순위: 항공권 (Aviasales)

**가입 정보**:
- URL: https://www.travelpayouts.com/en/aviasales/affiliate-program
- 가입 조건: 웹사이트 또는 앱 있으면 됨 (chaovietnam.co.kr 있음)
- 승인: 24~48시간

**수수료**: 
- 클릭당 약 $0.05 (CPC)
- 예약당 1.6~2% (티켓당 평균 $5~15)

**활용 방법**:
1. 검색 위젯 (Aviasales 제공) → 데일리 뉴스 이메일/웹 상단
2. 자동 검색 결과 PNG → "인천-호치민 오늘 최저가 35만원" 이미지로 표시
3. 클릭 시 Aviasales 검색 페이지로 (제휴 추적 자동)

**예상 월 수익**: 이메일 7,000 × 1% CTR × $0.05 = $35 (시작점)
실제 예약 발생 시 $100~500 추가 가능

**구현 작업**:
- Aviasales API 키 발급 후 일일 항공권 자동 검색
- 인천-호치민/하노이/다낭/하이퐁 노선 모니터링
- 최저가 + 변동 표시
- 이메일 PNG 자동 생성

### 2순위: 국제 송금 (Wise)

**가입 정보**:
- URL: https://wise.com/affiliates
- 승인: 1~2일

**수수료**:
- 신규 가입자 1명당 £10 (약 $13)
- 송금 첫 사용 시 추가 보너스

**활용 방법**:
1. 환율 위젯 옆 "지금 송금하면 OO원 절약" CTA
2. 데일리 뉴스 한국 환율 변동 시 "이번 주 송금하기 좋은 날" 콘텐츠
3. 가입 시 사용자에게 $X 보너스 (Wise가 지급)

**예상 월 수익**: 월 20명 가입 시 $260

**구현 작업**:
- 송금 시뮬레이터 위젯 (KRW → VND, 한국에서 베트남)
- 비교 표 (Wise vs 일반 은행 송금 수수료)
- 한 달에 1~2회 송금 가이드 콘텐츠

### 3순위: 호텔 예약 (Agoda)

**가입 정보**:
- URL: https://partners.agoda.com
- 승인: 즉시~3일

**수수료**: 예약 금액의 4~7%

**활용 방법**:
1. 베트남 여행 추천 콘텐츠 → 호치민·하노이·다낭·푸꾸옥 호텔 카드
2. 한국 출장/여행 콘텐츠 → 서울·부산 호텔
3. 데일리 뉴스 주말판에 "이주 호텔 특가" 섹션

**예상 월 수익**: 월 5건 예약, 평균 $300 × 5% = $75

**구현 작업**:
- Agoda 검색 위젯 또는 호텔 카드 자동 생성
- 데일리 뉴스 주 1회 호텔 섹션 자동 삽입

### 4순위: eSIM (Airalo)

**가입 정보**:
- URL: https://www.airalo.com/airmoney (Airmoney program)
- 또는 ShareASale를 통한 가입

**수수료**: 판매당 10~15%

**활용 방법**:
1. 베트남 여행 가는 한국 가족·지인 추천용
2. 한국 출장 가는 사용자에게 eSIM 추천
3. 위젯 또는 더보기 메뉴 "출장·여행 필수" 섹션

**예상 월 수익**: 월 30개 판매 × $10 × 12% = $36

### 5순위: 베트남 액티비티 (KKday + Klook)

**가입 정보**:
- KKday: https://www.kkday.com/en/affiliate
- Klook: https://www.klook.com/en/affiliate

**수수료**: 예약당 3~6%

**활용 방법**:
1. 주말 베트남 여행 콘텐츠 → 메콩 투어, 다낭 액티비티 카드
2. 한국 가족·지인 베트남 방문 시 추천용
3. 매월 1회 "이달의 추천 액티비티" 섹션

**예상 월 수익**: 월 $50~150

### 6순위: 글로벌 식자재 직구 (11번가 글로벌)

**가입 정보**:
- 11번가 제휴 마케팅 또는 쿠팡 파트너스 (한국 IP에서 가입 가능)

**수수료**: 2~5%

**활용 방법**:
1. "베트남에서 그리운 한국 식자재" 콘텐츠
2. 명절 전 한국 명절 음식 직구 가이드
3. 광고 카드와 분리해서 자연스럽게

### Phase 1 총합 예상

| 제휴 | 월 수익 (보수) | 월 수익 (낙관) |

| Aviasales | $50 | $300 |

## Phase 2: Phase 1 데이터로 협상 필요 제휴 접근 (30~60일)

Phase 1에서 **클릭 수, 전환률, 카테고리별 인기도 데이터**가 쌓이면 협상 필요한 곳에 접근 가능.

### 2-1. 베트남 거주자 대상 보험

**대상**: AXA, Allianz, Bao Viet, 삼성생명 베트남
**접근 방법**: 
- "월 7,000명 이메일 + 2,000 앱 도달 = 베트남 한인 도달률 X%" 미디어킷 제시
- Phase 1의 다른 제휴 데이터(예: Wise 가입 전환률)를 보여주며 "보험도 비슷한 전환률 기대" 협상
- 초기에는 일반 광고 형태 → 데이터 쌓이면 CPA 계약

**수수료 협상 목표**: 가입 1건당 $30~80

### 2-2. 한국 카드사 베트남 거주자 카드

**대상**: KEB하나, KB국민, 신한카드
**접근 방법**: 본사 제휴 마케팅 부서 직접 컨택
**제시 자료**:
- 베트남 거주 한국인 명확한 타겟팅 인프라
- 24년 잡지 신뢰 자산
- 데이터: 환율 위젯 클릭률 등

**수수료 협상 목표**: 신규 카드 발급 1건당 ₩30,000~50,000

### 2-3. 베트남 부동산 (vnkorlife 강화 또는 외부)

**대상**: 베트남 한국인 부동산 중개 업체 + Batdongsan
**현재 자산**: vnkorlife.com 부동산 페이지
**접근 방법**: 우리 자체 부동산 페이지 트래픽 데이터 + Phase 1 임대료 위젯 클릭률
**수익**: 거래 성사 1건당 큰 수수료 ($200~1,000)

### 2-4. 베트남 행정 서비스 (비자, 노동허가증)

**대상**: 베트남 한인 행정사 사무소 (호치민·하노이 각 5~10곳)
**접근 방법**: 직접 영업
**수수료**: 의뢰 건당 $20~50 또는 매월 정액 광고비

### 2-5. 베트남 의료 (한국인 진료 가능 병원)

**대상**: FV Hospital, Vinmec, 한국인 의사 있는 클리닉
**접근 방법**: 직접 영업
**수수료**: 환자 소개당 또는 월 광고비

### 2-6. 베트남 어학원/한국어 학원

**대상**: 한국인 자녀 대상, 베트남인 한국어 학원
**수수료**: 등록 1건당 1개월치 수강료의 X%

## Phase 3: 자체 데이터 자산 미디어킷화 (60~120일)

Phase 1+2에서 누적된 데이터를 **광고주 영업 무기**로 전환.

### 3-1. 통합 미디어킷 v2 작성

기존 [marketing/MEDIA_KIT_v1.md](../chao-vn-app/marketing/MEDIA_KIT_v1.md) 보강:
- 카테고리별 사용자 관심도 (위젯 클릭 데이터 기반)
- 시간대별 활성도
- 지역별 분포 (호치민 vs 하노이 vs 기타)
- 제휴 전환률 케이스 스터디

### 3-2. 광고주 패키지 상품화

- **베이직**: 잡지 광고 (기존)
- **디지털**: 잡지 + 데일리 뉴스 노출 (기존)
- **풀패키지**: 잡지 + 디지털 + 위젯 + 카테고리 페이지 + 제휴 추적
- **CPA 상품**: 신규 — 성과 기반 광고 (성공 시에만 결제)

### 3-3. 자체 비교/추천 서비스 출시

데이터가 충분히 쌓이면 자체 비교 서비스 시작:
- "송금 비교" — Wise vs 일반 은행 + 우리 제휴 링크
- "여행 비교" — Aviasales + Skyscanner + Trip.com 결과 비교
- "호텔 비교" — Agoda + Booking + Hotels.com 비교

각 비교 페이지에서 사용자가 클릭하는 곳에서 수수료 발생.
**이건 단순 제휴를 넘어 "씬짜오가 운영하는 서비스"가 됨.**

## Phase 4: 장기 — 자체 서비스화 (120일~)

### 4-1. 씬짜오 송금 (Wise 또는 자체 PG 연동)
중간 마진 수익 모델. 베트남 ↔ 한국 직접 송금 서비스.

### 4-2. 씬짜오 비자/행정 대행
신뢰 자산 활용. 사장님 인맥 + 한인 행정사 연결.

### 4-3. 씬짜오 카드/금융 상품
한국 카드사와 공동 발급 또는 자체 신용 카드.

### 4-4. 씬짜오 트래블
패키지 여행 직접 기획.
