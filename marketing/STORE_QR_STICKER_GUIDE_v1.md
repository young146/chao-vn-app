# 매장 QR 스티커 배포 가이드 v1

> 오프라인 → 앱 다리. 한인 식당·마트·미용실 매장 출입구·계산대에 부착하는 QR 스티커 운영 가이드.
>
> 작성: 2026-05-21
> 마케팅 활성화 방안 §3 액션 7 — 31~60일 단계
>
> 사용 안내: 이 문서는 운영 가이드입니다. 실제 인쇄용 디자인은 별도 (사용자 또는 디자이너가 Canva/Illustrator 작업).

## 1. 왜 매장 QR 스티커인가

**문제**: 베트남 한인은 호치민 ~8만 + 하노이 ~3만 정도의 *닫힌 시장*. 매스마케팅 비효율. 그러나 *매일 한인 식당·마트·미용실은 들름*. 이게 *가장 비용 효율적인 신규 사용자 접점*.

**해법**: 한인 매장 출입구·계산대·테이블에 *작은 QR 스티커* 부착. 손님이 식사 기다리며 또는 결제 시 자연스럽게 스캔 → 앱 설치 또는 콘텐츠 노출.

**광고주에게 줄 가치**:
- 매장 무료 노출 (씬짜오 앱 상세 페이지 + 쿠폰)
- 손님 추적 가능 (어느 매장에서 온 사용자인지 GA4 로 측정)
- 잡지 광고주에게 *우선 무료 제공* — 정기 광고주 혜택

---

## 2. QR 가리키는 URL

**v1 (간단)**: 모든 매장 동일 URL
```
https://chaovietnam-login.web.app/go/app?utm_source=offline&utm_medium=qr_sticker&utm_content=store
```

→ OS 자동 분기 (Android/iOS) → 앱 설치 페이지

**v2 (매장별 추적, 향후)**: 매장별 코드
```
https://chaovietnam-login.web.app/go/app?utm_source=offline&utm_medium=qr_sticker&utm_content=store_${storeCode}
```

→ GA4 에서 *어느 매장*이 가장 다운로드 유도가 좋은지 측정 가능

## 3. 스티커 디자인 가이드

### 사이즈 권장
| 용도 | 사이즈 | 위치 |
|---|---|---|
| **테이블 스티커** | 8cm × 8cm | 식당 테이블 상판 (코팅) |
| **계산대 스티커** | 10cm × 10cm | 카운터 시야 정면 |
| **출입구 스티커** | 15cm × 15cm | 입구 유리문 |

### 색·톤
- 배경: 흰색 (가독성) 또는 씬짜오 브랜드 색 (`#FF6B35` 주황)
- QR 코드: 검정 (스캔 안정성 — 컬러 QR 은 인식 실패율 ↑)
- 글자: 검정 또는 진한 회색

### 필수 요소 (한국어 + 베트남어 병기 권장)
```
[QR 코드]

📱 씬짜오베트남 앱
베트남 한인 24년의 신뢰

· 채용 정보
· 부동산 매물
· 한국 식당 쿠폰
· 매일 한인 뉴스

스캔 → 앱 설치 (iOS·Android 자동)
```

베트남어 부제 (선택): "Ứng dụng cộng đồng Hàn Quốc tại Việt Nam"

## 4. 매장 입점 협력 제안 — 광고주에게 보낼 한 줄

> "**씬짜오 잡지 광고주에 한해 무료** — 매장 QR 스티커 (8×8cm 5장) 배포 + 매장 정보 앱 등록. 손님 유입 추적 리포트 월간 제공."

이 한 줄이 **잡지 광고주 → 디지털 전환 미끼**. 광고 1건당 부가 혜택으로 매장 노출 + 추적.

### 광고주 안내 메일 템플릿
```
[씬짜오베트남] 매장 무료 QR 스티커 안내

안녕하세요, [광고주명] 사장님,

씬짜오베트남 잡지를 통한 광고에 더해
*무료 매장 QR 스티커* 를 함께 제공해드립니다.

✨ 제공 내용
· 8×8cm 5장 — 테이블/계산대 부착용
· 씬짜오 앱 안 매장 상세 페이지 1년 무료 등록
· 손님 스캔 수·앱 설치 유도 수 월간 리포트

비용: 무료 (씬짜오 잡지 광고주 한정)
배포: 2주 이내 배송 (호치민·하노이·다낭)

회신 주시면 디자인 시안 + 매장 등록 안내 보내드립니다.

— 씬짜오베트남 마케팅팀
```

## 5. 운영 절차

1. **타겟 광고주 선정** — 잡지 정기 광고주 중 한인 매장 운영자 (식당·마트·미용실·여행사 등)
   → 예상 규모: 🔴 TODO 잡지 광고주 풀 중 _____곳
2. **이메일 발송** — 위 템플릿 → 응답률 측정
3. **스티커 인쇄** — 신청 매장당 5장 × 사이즈 3종 (테이블·계산대·출입구)
   → 인쇄 단가 추정: 🔴 TODO (예: 스티커 1장당 5,000동? 컬러 인쇄 + 코팅 포함)
4. **매장별 코드 배정 (v2)** — 등록 매장에 고유 코드(예: HCM_001, HCM_002...) 부여
5. **배송** — 본사 → 매장 (호치민/하노이/다낭별 라우트)
6. **GA4 측정** — 월말 매장별 스캔 수·설치 수 리포트 → 광고주 발송

## 7. 위험·한계

- **베트남 사람의 QR 스캔 관행 낮음** — 한국·중국 대비 베트남은 QR 스캔 비율이 낮은 편. 한인 손님 중심 타겟
- **스티커 훼손/제거** — 매장이 청소·리모델링 시 떼어낼 수 있음. 정기 점검 필요
- **광고주 매장 운영자 협조** — 일부 매장은 "내 매장에 다른 광고 X" 정책. 무료 제공이라도 거부 가능
