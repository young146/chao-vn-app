Chao Vietnam — 6탭 디자인 비교

실제 모바일 너비에서 어떻게 보이는지 비교해보세요. 탭을 눌러 선택 상태도 확인 가능.

📝 평가 기준

  • Fit — 375px (가장 좁은 iPhone)에서 "이웃사업"이 잘리지 않고 들어가는가
  • 스캔 속도 — 사용자가 원하는 탭을 얼마나 빨리 찾을 수 있는가 (아이콘이 도움이 됨)
  • 다국어 대응 — 베트남어("Việc làm", "Bất động sản")로 바꿨을 때도 들어가는가
  • 유지보수 — 현재 코드(tabBarIcon + tabBarLabel)에서 변경 범위가 얼마나 큰가

💡 구분선 4종 비교 노트

  • G (점선): 디버그 시각화용. 실제 앱에서 쓰면 "아직 디자인 중" 느낌
  • G1 (실선 #e5e5e5): 각 탭이 버튼처럼 명확. 좀 전통적
  • G2 (옅은 실선 #f0f0f0): 구조감은 있지만 거의 무의식 수준 — 추천
  • G3 (선 없음): 가장 미니멀하나 탭 경계가 흐려짐
  • 구현 난이도: 모두 CSS 한 줄 차이 (React Native도 tabBarItemStyle에 borderRightWidth: 1 + borderRightColor 추가면 끝)
